## Functions

### Math

**[Clamp](https://github.com/theoplawinski/utils/blob/main/math/clamp.ts)**

Clamps a value between an upper and lower bound

```javascript
import { clamp } from "./math/clamp";

const value = clamp(100, 0, 25); // 25
```
